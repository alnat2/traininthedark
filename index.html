Introduction

Train in the dark is inspired by the Code in the dark competition. But just for training )

Main differences:

  • You can change the duration
  • You can use Emmet
  • You can copy-paste text

Instructions

To start a training, you need to do a few things:

  1. Upload your assets (download example here)
  2. Copy text and assets` names from instructions tab
  3. Reference tab for... a reference image )
  4. Use the filename as the path to the image
  5. Hit the start button
